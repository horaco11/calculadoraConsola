#Notas

Ejecutar npm install para instalar dependencias necesarias.

Flags de la aplicacion.
```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Define la base de la tabla de multiplicar. Ejemplo: Tabla del 5
                                                             [number] [required]
  -l, --listar   Utilice este flag para mostrar la tabla por pantalla.
                                                      [boolean] [default: false]
  -h, --hasta    Utilice este flag para indicar hasta que multiplicador crea la
                 tabla.                                   [number] [default: 10]
```
